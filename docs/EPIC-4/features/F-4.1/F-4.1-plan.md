## üé® Visual Identity (Aesthetic & Style)
- **Base Atmosphere**: Consistent with the **Expedition Mission Briefing** theme defined in `outdoor-profile-prd.md`.
- **Selected Background: The Heritage Path**: Uses `TopographyBackground` (flickering topo lines) + `ScanLine` overlay. This maintains 1:1 visual continuity with the `/career` route.
- **Color Palette**:
    - **Primary Base**: `bg-night-summit` (`#0D1B2A`).
    - **Globe Highlights**: `text-alpine-blue` (`#82B7F5`) for the neon glow and orbital markers.
    - **Interface Metadata**: `text-glacier-steel` (`#7A8FA3`) for secondary data.
- **Typography**: 
    - **Oxanium**: Tactical labels and technical data.
    - **Primary Header**: `[ ADVENTURES ]` in white bold Oxanium, matching the global brand identity.
- **Shared Expedition Components**: Uses `TopographyBackground`, `ScanLine`, and **`ExpeditionFooter`** (Contact icons) to ensure the interface feels part of the same operating system.
- **The Sequential Boot-up**: 
    - **UI First**: The interface framing, sidebar headers, footer contacts, and technical readouts initiate a sequential "typing" and "glitch" entrance.
    - **Delayed Recon**: The Globe only begins its entry sequence **after** the UI has reached its stable state (approx. 1.2s delay).
    - **Classy Globe Entry**: A smooth `opacity: 0 to 1` combined with a subtle `scale: 0.95 to 1.0` spring animation.
