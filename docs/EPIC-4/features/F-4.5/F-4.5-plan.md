### 3. UI Component Enhancement (`AdventureBriefingModal.tsx`)
The modal currently uses a placeholder for the visual section. We will:
- **Image Container**: Maintain the **1:1 Square** aspect ratio.
- **Scanning Overlay**: Add a CSS-based "laser line" that moves vertically across the image to simulate a data scan.
- **Loading State**: Show a technical "DECODING_IMAGE..." text while the image is loading.
- **Glitch Effect**: Apply a subtle Framer Motion glitch (sudden scale/opacity shifts) when the image first appears.

### 4. Technical Aesthetics (CSS)
Add specific utility classes or styled-components for:
- `.recon-scan-line`: A horizontal semi-transparent line with a glow effect.
- `.intel-frame`: A thin border with "corner-brackets" around the image.
