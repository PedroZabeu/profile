# Adventure Exploration: AI Image Prompts

This document contains specialized prompts for generating the "Visual Exploration" assets for the Global Archive (EPIC 4). Use a high-end AI image generator (Flux, Midjourney, etc.) for best results.

## Global Aesthetic Style
**Style Descriptor**: Majestic, cinematic, high-resolution professional outdoor expedition photography. Shot on Phase One XF, 100MP. Sharp focus, dramatic natural lighting (golden hour or harsh alpine sun), high contrast, slightly desaturated shadows with rich highlights. Professional color grading, national geographic quality.
**Dimensions**: Aspect Ratio 1:1 (Square). For Midjourney/Flux use `--ar 1:1`.
