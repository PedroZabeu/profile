## ðŸ§± Key Components

### 1. `AnimatedClouds`
- **Purpose**: A dedicated component to manage the rendering and horizontal movement of cloud clusters.
- **Location**: `components/animated-showcase/background/AnimatedClouds.tsx`
- **Features**:
    - Uses an array of cloud images (from `/public/images/clouds/`).
    - Implements unidirectional right-to-left movement using Framer Motion.
    - Configurable properties per cloud: `src`, `top`, `duration`, `delay`, `scale`, and `opacity`.
    - **Looping**: Infinite loop with random-looking offsets to ensure a continuous sky.

### 2. `AnimatedBackground` (Update)
- **Integration**: The `AnimatedClouds` component will be layered over the static mountain image but under the (future) logo and dots.
- **Z-Index Strategy**: 
    - Mountain Base: `z-0`
    - **Clouds Overlay: `z-10`**
    - Details/Content: `z-30`
