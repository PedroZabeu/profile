### 1. `AnimatedLogo`
- **Purpose**: A specialized wrapper for the brand logo that manages its entrance timing and parallax behavior.
- **Location**: `components/animated-showcase/logo/AnimatedLogo.tsx`
- **Features**:

## ðŸ“‹ Implementation Steps

1. **Component Scaffolding**:
   - Create the directory `components/animated-showcase/logo/`.
   - Create `components/animated-showcase/logo/AnimatedLogo.tsx`.

2. **Logo Implementation**:
   - Import `Logo` from `@/components/brand/Logo`.
   - Implement the entrance animation using `motion.div`.
   - Pass `mousePosition` and `logoY` (scroll transform) as props to maintain the parallax effect.
   - Use `filter drop-shadow-[0_20px_50px_rgba(0,0,0,0.8)]` to match the original showcase depth.

3. **Page Integration**:
   - Update `app/animated-showcase/page.tsx`.
   - Calculate `logoY` using `useTransform(scrollY, [0, 500], [0, -40])`.
   - Place `<AnimatedLogo />` within the sticky container.
   - Ensure it is centered (`flex items-start justify-center pt-12`).

4. **Refinement**:
   - Verify the 0.3s delay feels rhythmically correct after the page load.
   - Ensure the logo remains interactive (pointer-events-auto) while its container is `pointer-events-none`.
