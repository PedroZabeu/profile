## üìã Implementation Steps

### 1. Component Creation
- Create directory: `components/animated-showcase/scroll-button/`
- Create `AnimatedScrollButton.tsx` with:
    - Simplified "CONTACT" text
    - Oxanium font (semibold)
    - Pure text glitch animation (no bounce/pulse)
    - No background or border
    - Scroll functionality (same as showcase)

### 2. Animation Implementation
```javascript
const glitchBounceEntrance = {
  initial: { opacity: 0, scale: 0.8, y: 20 },
  animate: { 
    opacity: 1, 
    scale: 1, 
    y: 0,
    transition: {
      delay: 2.3, // Start after dots complete at 2.3s
      duration: 0.8,
      ease: [0.68, -0.55, 0.265, 1.55] // bounce easing
    }
  }
};

const glitchKeyframes = {
  scale: [1, 1.1, 0.9, 1.05, 1],
  opacity: [0, 0.8, 1, 0.9, 1],
  filter: ["blur(2px)", "blur(0px)", "blur(1px)", "blur(0px)"],
  transition: { duration: 0.8 }
};
```

### 3. Styling Enhancements
- Use `font-oxanium` and `tracking-[0.3em]`
- Remove all backgrounds, borders, and backdrops
- Implement continuous subtle jitter glitch
- Add occasional color-shift text glitch overlays

### 4. Integration
- Update `app/animated-showcase/page.tsx`
- Add scroll button to the sticky container at `z-25`
- Ensure proper layering: below dots (`z-20`) but above content
- Implement same smooth scroll behavior as `/showcase`

### 5. Parallax Consistency

- Use `useTransform(scrollY, [0, 500], [0, -30])` for vertical movement
- Maintain consistent movement with page scroll
