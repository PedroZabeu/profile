# Betting Management - Visual e Conteúdo

## 🎯 Visão Geral

Sistema de gestão operacional para contas de betting com dashboard analítico e tabelas detalhadas. Foco principal no controle de saldos e lucros individuais e consolidados.

## 🎨 Elementos Visuais do Demo

### **Cores e Estilo**
- **Background**: `bg-background` (escuro)
- **Cards**: `bg-card` com `border`
- **Textos**: `text-foreground` e `text-muted-foreground`
- **Destaques**: `text-cv-accent` (teal) para valores positivos
- **Título**: Ícone 📊 + "Dashboard" simples

### **Ícones e Indicadores**
- **Dashboard**: 📊 (único ícone no topo)
- **Valores**: Destaque com cor teal para positivos
- **Contas**: Lista simples sem status complexos

### **Animações Sugeridas**
- **Cards**: Hover sutil com `scale-102`
- **Valores**: Fade-in suave ao carregar
- **Layout**: Sem animações complexas para manter limpo

## 🌐 Internacionalização do Demo

### **Elementos Localizados**
- **Títulos**: "Dashboard" vs "Painel"
- **Colunas**: "Balance" vs "Saldo", "Profit" vs "Lucro"
- **Status**: "Active" vs "Ativa", "Paused" vs "Pausada"
- **Métricas**: Formatação de moeda (R$ vs $)

### **Adaptação Visual**
- **Português**: Formato brasileiro (R$ 1.234,56)
- **Inglês**: Formato internacional ($1,234.56)
- **Cores**: Mantidas em ambos os idiomas
- **Layout**: Responsivo em ambos os idiomas
