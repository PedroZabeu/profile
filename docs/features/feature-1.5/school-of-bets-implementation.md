# School of Bets - Visual e Conteúdo

## 🎯 Visão Geral

Plataforma educacional completa para apostadores, com sistema de aulas, modules, progresso e certificação. Foco principal no ensino estruturado e trilha de aprendizagem personalizada.

### **Identidade do Projeto**
- **Nome**: School of Bets
- **Categoria**: Plataforma Educacional
- **Foco**: Sistema de Aulas + Progresso + Certificação

### **Estratégia de Reaproveitamento**

#### **Prioridade 1: Reaproveitar 100%**
- Sistema de idiomas (`Language`, `Record<Language, string>`)
- Sistema de cores (`ColorTheme`, `COLOR_SYSTEM`)
- Sistema de animações (`MotionProps`)
- Estrutura de dados (`ProjectData`)
- Componentes UI base (`CardProps`, `ButtonProps`)

#### **Prioridade 2: Estender Mínimo**
- Criar apenas tipos específicos do domínio educacional
- Adicionar classes CSS apenas para estilos únicos (progresso, módulos)
- Manter padrão de nomenclatura existente

#### **Prioridade 3: Seguir Padrões**
- Usar `emerald` como cor do projeto (diferente de `teal` e `violet`)
- Manter estrutura de dados consistente com projetos existentes
- Aplicar sistema de animações existente
- Seguir convenções de nomenclatura `school-bets-*`

### **Benefícios do SSOT**

1. **Manutenibilidade**: Mudanças no sistema afetam todos os projetos
2. **Consistência**: Cores, animações e estrutura unificados
3. **Performance**: Menos código duplicado
4. **Escalabilidade**: Fácil adicionar novos projetos seguindo padrão
