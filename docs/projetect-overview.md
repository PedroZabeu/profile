# Project Overview: CV Online Interativo

## Visão Geral

Um CV online em formato de **storytelling interativo** para o mercado de startups digitais. O foco é mostrar **capacidade de criar soluções** e **transformar ideias em produtos**, não listar títulos e datas.

## Interações Sugeridas

### Microinterações

```
+-------------------------------------------------------+
|                                                       |
|   1. HOVER nos cards de portfolio                     |
|      -> Leve zoom + sombra + preview rápido           |
|                                                       |
|   2. SCROLL na timeline                               |
|      -> Cards aparecem com fade suave                 |
|      -> Ponto atual fica highlighted                  |
|                                                       |
|   3. CLICK nos skills                                 |
|      -> Expande com contexto de onde usei             |
|                                                       |
|   4. TRANSIÇÕES entre páginas                         |
|      -> Slide suave, não reload                       |
|                                                       |
+-------------------------------------------------------+
```

## Princípios de Design

1. **Mobile-first** -> Maioria vai acessar pelo celular
2. **Fast loading** -> Primeiras impressões importam
3. **Scannable** -> Ninguém lê CV, pessoas escaneiam
4. **Show, don't tell** -> Screenshots > descrições
5. **One CTA per section** -> Guiar sem confundir
