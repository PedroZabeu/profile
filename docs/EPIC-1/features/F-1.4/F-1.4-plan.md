## Detailed Plan for F1.4: Projects CTA Path

### Objective
Implement a Projects CTA on the /showcase page as a visual hook to the projects timeline, featuring a glowing/pulsing dot with rings and ripple animation in the mountain image that reveals a modal with Pedro's projects summary in combat briefing style.

### Key Components
- **ProjectsCTA.tsx**: Interactive dot in mountain image
- **ProjectsModal.tsx**: Animated modal with typed projects hook

### Implementation Steps
1. Position glowing/pulsing dot with rings and ripple animation (Alpine Mist #F1F5F9) in the top-left quadrant of the mountain image, closer to the center (midway between the center of the top-left quadrant and the center of the image)
2. Add hover effect: display "Projects" text in Alpine Mist color
3. Implement click handler to open modal with scale-in animation from dot to screen center
4. Create modal: square card, thin white borders, 95% white transparent background
5. Add typing animation for projects summary text using Oxanium font (Google Fonts)
6. Integrate Framer Motion for animations, including ripple animation for the dot
7. Ensure responsive design and accessibility

### Modal Content Details

#### Layout Structure
- **Header:** Title at top, left-aligned with 30px margin from left, 20px margin below.
- **Description:** Below title, left-aligned with 30px margin from left, 15px margin below.
- **Highlights:** Bullet list below description, left-aligned with 30px margin from left, 20px margin below.
- **CTA:** Left-aligned button at bottom with 30px margin from left, 20px margin above.

#### Typography
- **Title:** "PROJECTS" (all uppercase), Oxanium font (Google Fonts), 28px font-size, #F1F5F9 color (Alpine Mist), bold weight, military-style caps.
- **Description:** "Full-stack web applications builder. From PRD to fully functionning apps", Oxanium font, 18px font-size, #F1F5F9 color, regular weight, centered.
- **Highlights:** Bullet points: "Betting Mgmt", "Stakely", "School of Bets". Oxanium font, 16px font-size, #F1F5F9 color, with • bullet symbol, left-aligned with 10px indent.
- **CTA Button:** "Details" text, Oxanium font, 20px font-size, #F1F5F9 color, no border/background, hover effect: opacity 0.7 transition over 0.3s.

#### Spacing and Styling
- **Modal Dimensions:** Square card, width/height 400px (responsive: 90% max-width on mobile).
- **Padding:** 30px internal padding around all content.
- **Background:** 95% white transparent (rgba(255,255,255,0.95)).
- **Borders:** Thin white borders (1px solid #F1F5F9).
- **Margins:** 20px between sections (title-desc, desc-highlights, highlights-cta).

#### Animations
- **Scale-in:** Modal scales from dot position to screen center, duration 0.5s, ease-out using Framer Motion.
- **Dot Disappearance:** When modal opens, glowing/pulsing dot fades out (0.3s).
- **Ripple Animation:** Pulsing dot with expanding rings/ripple effect using Framer Motion.
- **Text Typing:** Title appears instantly, description types out character-by-character (100ms per char), highlights fade in sequentially (0.3s each, 0.2s stagger), CTA fades in last.
- **Hover CTA:** Opacity transition on button hover, 0.3s ease.
- **Closing:** Click outside modal to close with fade-out animation (0.3s), dot reappears, fading in.
- **Overall:** Military briefing feel with typed text, using Framer Motion for smooth transitions.
