# Feature Plan: F-3.1 Stage 3 - Interactive Milestone CTA

This stage transforms the static career log into an interactive dashboard where each milestone acts as a Call-to-Action (CTA) to access deep experience logs.

- **Milestone Interaction:** Each `CareerEntry` becomes a fully interactive zone.
- **Hover Elegance:** A "System Access" aesthetic. Hovering on an entry should feel like highlighting a row in a technical terminal.
- **Visual Cues:**
    - **Row Highlight:** A very subtle `bg-white/5` or `bg-glacier-steel/5` background fade.
    - **Cursor:** `cursor-pointer`.
    - **Interaction Pointer:** A minimalist "EXPAND" icon or a technical chevron that slides into view upon hover.
- **Action:** Clicking an entry triggers the `ExpeditionLog` modal/view (to be detailed in Stage 3 implementation).
